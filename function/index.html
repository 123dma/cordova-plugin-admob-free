    public setOptions(options: Object, successCallback: function(), failureCallback: function()) source 

    import {setOptions} from 'cordova-plugin-admob-free/admob'
    this function was deprecated. since version 0.6

    Set options.

    Params:

    NameTypeAttributeDescription
    options Object
    options.publisherId string
    options.interstitialAdId string
    options.bannerAtTop boolean
    • optional
    • default: false

    Set to true, to put banner at top.
    options.overlap boolean
    • optional
    • default: true

    Set to true, to allow banner overlap webview.
    options.offsetTopBar boolean
    • optional
    • default: false

    Set to true to avoid ios7 status bar overlap.
    options.isTesting boolean
    • optional
    • default: false

    Receiving test ad.
    options.autoShow boolean
    • optional
    • default: true

    Auto show interstitial ad when loaded.
    options.forChild boolean | null
    • optional
    • default: null

    Default is not calling tagForChildDirectedTreatment. Set to "true" for tagForChildDirectedTreatment(true). Set to "false" for tagForChildDirectedTreatment(false).
    options.forFamily boolean | null
    • optional
    • default: null

    Android-only. Default is not calling setIsDesignedForFamilies. Set to "true" for setIsDesignedForFamilies(true). Set to "false" for setIsDesignedForFamilies(false).
